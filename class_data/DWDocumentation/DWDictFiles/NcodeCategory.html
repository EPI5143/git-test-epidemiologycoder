Table: NcodeCategory

Description: Code Categories for lookup values from other columns

Column Name

Description

Format

Identified

Code Category (*=mandatory)

cdcWCT

The date/time the entry was created in the DW. This does not reflect activity in the source system, just when the row was created in the DW. In some instances, updates are performed as delete / insert making this field's usefulness more limited.

datetime

No

cdcWUT

The date/time the entry was last updated in the DW. This does not reflect activity in the source system, just when the row was last updated in the DW. In some instances, updates are performed as delete / insert making this field's usefulness more limited.

datetime

No

cdcWUU

The userid or user who last updated the entry. This can be 'ArchiveLoad' to reflect initial bulk loading of the DW or something like 'bfAllergy:31:3373:' which identifies an ETL load process used to update information in the DW.

varchar(250)

No
